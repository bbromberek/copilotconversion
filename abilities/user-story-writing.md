# User Story Writing

## User Story Template

As a [type of user], I want [goal] so that [reason].

## Example of a User Story

As a frequent traveler, I want to easily book a flight online so that I can save time and avoid long queues at the airport.

## Acceptance Criteria
- The user must be able to search for flights by origin, destination, and date.
- The user should receive confirmation of the booked flight via email.
- The user can view and manage their bookings through their account.
